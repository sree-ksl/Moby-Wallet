A Tron wallet for android.

## Refrence
- https://github.com/tronprotocol/wallet-cli

## Required.
 - android 4.1

## Features

- Create wallet
- Import wallet
- Send tron & token
- Receive tron & token
- My address QR code
- My tron balance
- My tokens
- My transfer history
- Vote representative
- Participate token
- Block explorer
- Token list
- Node list
- Representative list
- Tron market listing & price
